# About Me

嗨，我是 Ruby Lin，一位 **Frontend-strong 的 Full-stack Engineer**。

我有 5 年以上電商與 App 開發經驗，前期主要投入 React、React Native 與 TypeScript 的商業系統開發；隨著對系統整體流程、資料流與跨部門協作的理解加深，近一年開始延伸到後端領域，使用 **Python / Django / PostgreSQL** 實作商品匯入、庫存同步、POS 系統整合等商業模組。

我目前仍持續參與一線功能開發，同時也在團隊中負責部分技術方案討論、任務拆解與 code review，協助對齊實作方式與程式碼品質。我的下一步目標，是把自己從前端主力角色，逐步轉成 **更完整的全端／後端導向工程師**。
