# GO-Ticket

## 專案定位

GO-Ticket 是一個用來展示我後端與全端實作能力的 side project，主題是活動售票系統。

## 我想透過這個專案展現的能力

- RESTful API 設計
- PostgreSQL 資料表與關聯建模
- 身分驗證與權限流程
- Docker 化開發環境
- 部署與 CI/CD 基礎流程

## 技術棧

- Backend: Node.js / Express
- Database: PostgreSQL
- Auth: JWT
- Infra: Docker / Nginx / AWS EC2 / S3

## 功能範圍

- 活動管理
- 訂單流程
- 票券管理
- 使用者驗證

## 這個專案的價值

相較於單純做頁面，我希望這個專案能更完整呈現我對於資料流、商業流程、驗證機制與部署流程的理解，也讓我的履歷不只停留在前端開發，而是能具體展示往後端與全端延伸的能力。
